@startuml

actor  Пользователь
boundary Браузер
entity Сервер

Пользователь -> Браузер : Вводит google.com
Браузер -> Браузер : Проверяет список HSTS
Браузер -> Браузер : Проверяет домена в своем DNS-кэше
Браузер -> Браузер : Запрашивает поток TCP-сокета
Браузер -> Сервер : Запрашивает TLS handshake
Сервер -> Браузер : Отвечает на TLS handshake
Браузер -> Сервер : Отправляет HTTP-запрос
Сервер -> Браузер : Отправляет ответ
Браузер -> Браузер : Парсит HTML
Браузер -> Браузер : Интерпретирует CSS
Браузер -> Браузер : Рендерит страницу
Браузер -> Пользователь : Отображает страницу

@enduml
